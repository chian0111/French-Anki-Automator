# 🇫🇷 Anki French Automator: Aesthetics & Logic

這是一個結合 Python 自動化、gTTS (Google Text-to-Speech) 與法式美學設計的 Anki 卡片生成工具。

## ✨ 專案亮點
* **自動化流程**：一鍵將法文單字轉換為具備標準發音與優雅排版的 Anki 卡片。
* **莫蘭迪美學設計**：採用 `Playfair Display` 襯線體與 `莫蘭迪綠` 配色，並在 CSS 中植入隱喻式的法國國旗元素，打造如藝術館導覽牌般的視覺體驗。
* **技術突破**：
    * 解決了 `genanki` 與最新版 Python 的 `write_to_file` 兼容性問題。
    * 實現了基於 `gTTS` 的自動語音封裝技術。

## 🛠️ 技術棧
* **Language**: Python 3.12
* **Libraries**: `genanki`, `gTTS`, `os`
* **Styling**: HTML5, CSS3 (Neumorphism & Flexbox)

## 📖 開發心路歷程
這個專案誕生於 2026 年初的靜養期間。當時我正準備著 CFA 與 GMAT 的挑戰，同時沉浸在法文與藝術的世界中。我意識到學習不應只是枯燥的填鴨，更可以是一場視覺與聽覺的饗宴。

透過這個工具，我將會計專業的嚴謹邏輯與對法式生活的熱愛結合，實踐了「讓數位工具為感性生活服務」的理念。
